F1 - Open/Closes Help

Menus:
ESC - Guess...
Q - Open Object Selection
P - Open Proerty Menu 
	Context Specific: When Object selected, it will show the object properties.
			Otherwise it will show the Level properties.

When a button has brackets then its most likely a keybind.

Controls:
Left Click - (Un-)Selects a object and accepts modes.
W - Move Mode (Press Shift to move the center)
	Arrow Keys can be used to place the object more specificly
S - Scaling Mode
R - Rotate Object
F2 - Start test level

F1 - Open/Closes Help

Menus:
Q - Open Object Selection
P - Open Proerty Menu 
	Context Specific: When Object selected, it will show the object properties.
			Otherwise it will show the Level properties.

Controls:
Left Click - (Un-)Selects a object and accepts modes.
W - Move Mode
S - Scaling Mode
R - Rotate Object
F5 - Start test level
